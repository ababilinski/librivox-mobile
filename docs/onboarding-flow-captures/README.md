# Onboarding Flow Captures

Device screenshots of the LibriVox Mobile onboarding flow.

Captured on a connected Pixel 9 debug install after clearing only the onboarding profile state.

| File | Description |
| --- | --- |
| `00-welcome.png` | Welcome screen with the LibriVox Mobile setup intro and primary Get started action. |
| `01-permissions-default.png` | Permissions step with optional notification and nearby-device permissions enabled. |
| `02-permissions-optional-off.png` | Permissions step after turning both optional permissions off. |
| `03-languages-default.png` | Language step with all available LibriVox languages selected. |
| `04-languages-none-selected.png` | Language step after choosing None, showing the validation message. |
| `05-languages-all-selected.png` | Language step after choosing All again. |
| `06-languages-more-list.png` | Language step after using More to reveal the rest of the language list and the Next action. |
| `07-downloads-default.png` | Downloads and cache step with default manual-download settings. |
| `08-downloads-current-chapter.png` | Downloads step after selecting Current chapter pre-download. |
| `09-downloads-cache-section.png` | Downloads step scrolled to the search and browsing cache setting. |
| `10-downloads-cache-off.png` | Downloads step after turning the search cache off. |
| `11-onboarding-complete-home.png` | Home screen after finishing onboarding. |
